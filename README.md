# BAND//WIDTH

**A lateral platformer runner about WISP telecom technicians doing parkour on real infrastructure across Latin American and American cities.**

Hollow Knight × Rick & Morty style. Built entirely in Python + Pygame with procedural art and audio — zero external assets.

## Requirements

- Python 3.10+
- pygame
- numpy

## Installation

```bash
pip install pygame numpy
python bandwidth_game.py
```

## Controls

| Key | Action |
|---|---|
| WASD / Arrow Keys | Move |
| SPACE | Jump / Double Jump |
| SHIFT | Sprint |
| E | Interact / Install Antenna |
| Q | Special Ability |
| ESC | Pause |
| F | Toggle Fullscreen |
| TAB | Skip Dialog / Cutscene |

## Characters

| Character | Unlock | Special |
|---|---|---|
| Rico | Starter | Cable Whip — fiber lasso to reach far poles |
| Ing. Vero | Beat CDMX | Spectrum Scan — freeze jammers for 3s |
| Don Aurelio | Reach Level 15 | Señal Fantasma — ghost relay antenna |
| MorXel | Beat the game | Reboot Reality — reset all obstacles |

## Cities (30 Levels)

1. Monterrey (1–3)

4. Rio de Janeiro (10–12)
5. São Paulo (13–15)
6. Dallas (16–18)
7. Bogotá (19–21)
8. Buenos Aires (22–24)
9. Miami (25–27)
10. Final Showdown (28–30)

## Easter Eggs

Type in the main menu:
- **WISPA** → +25% speed boost
- **MIMOSA** → Perfect signal on all installs
- **CAMBIUM** → Aurelio helper NPC appears
